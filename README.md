# Magic 8 Ball Project
Main project for Skillcrush 102 project. This is an interactive magic 8 ball web page.

## Tech Stack
- HTML
- CSS
- JavaScript
- jQuery

## Instructions
1. Clone or download zip file. 
2. Open index.html file in your browser. 
3. Click the Ask Anything button and ask your question. Then wait and see what your answer is.
